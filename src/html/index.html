Geocaching hints and tips

  • Caches are brightly coloured (red, blue or yellow) cable ties (photo example of the cable ties (this is a PNG link)) - these will be hiding within the 3 metre square (see the what3words map for a more accurate position of where the cache is)
  • If you're able to, carry a bin bag and pick up any litter you find along your route when searching for geocaches (if you're a young person in Scouting, this may count towards an activity or challenge badge!)
  • Once you've found the cache - enter the 5 digit code into this web-app to verify that you've found it (this will be viewable on the Found caches page)
  • Make sure that you leave the cache as you found it - we want as many people to be able to find them as possible
  • If you're struggling to find a cache, seen a cache that is damaged, or think we've got the what3words location slightly wrong, please email us at inbox@geoscout.uk
